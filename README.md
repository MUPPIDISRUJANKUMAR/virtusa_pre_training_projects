# 🚀 Virtusa Pre-Training Projects

This repository contains three domain-based projects developed as part of pre-training preparation. These projects demonstrate practical implementation of concepts in Java, Python, and SQL.

## 📌 Projects Overview

### 1️⃣ Java Project – Online Quiz & Assessment System

#### 📖 Problem Statement
Educational platforms need systems to conduct quizzes and evaluate users efficiently.

#### 🎯 Objectives
- Create a system where users can take quizzes
- Automatically evaluate and display results

#### ✨ Key Features
- Admin can add questions and answers
- Multiple-choice quiz system
- Timer-based quiz
- Auto score calculation
- Result summary with performance analysis

#### 🛠️ Technologies Used
- Core Java (OOP concepts)
- File Handling / MySQL (optional)
- Optional GUI using Swing/JavaFX

#### ✅ Outcome
An interactive quiz system useful for learning platforms or mock tests.

### 2️⃣ Python Project – Smart Expense Tracker with Insights

#### 📖 Problem Statement
Many individuals struggle to track daily expenses and understand spending patterns.

#### 🎯 Objectives
- Record daily expenses (date, category, amount, description)
- Categorize spending (Food, Travel, Bills, etc.)
- Generate monthly summaries and insights

#### ✨ Key Features
- CLI or simple GUI input system
- Data storage using CSV or JSON
- Monthly expense summary
- Category-wise breakdown (pie chart using matplotlib)
- Detect highest spending category

#### 🛠️ Technologies Used
- Python
- CSV / JSON
- Matplotlib

#### ✅ Outcome
A tool that helps users understand where their money goes and suggests areas to reduce spending.

### 3️⃣ SQL Project – Online Retail Sales Analysis Database

#### 📖 Problem Statement
Retail businesses generate huge sales data but lack structured insights.

#### 🎯 Objectives
- Create a relational database for an online store
- Store customer, product, and order data
- Extract meaningful insights using SQL queries

#### 🗂️ Database Tables
- Customers (customer_id, name, city)
- Products (product_id, name, category, price)
- Orders (order_id, customer_id, date)
- Order_Items (order_id, product_id, quantity)

#### ✨ Key Tasks
- Find top-selling products
- Identify most valuable customers
- Monthly revenue calculation
- Category-wise sales analysis
- Detect inactive customers

#### 🛠️ Technologies Used
- SQL (MySQL / Oracle / PostgreSQL)

#### ✅ Outcome
A structured database with optimized queries that provide actionable business insights.
